Applicable Standards (5)

API — 2 standards

API 1104 Welding of Pipelines and Related Facilities
Edition: 22nd Edition (2021) Region: USA, Global
Welding procedures, welder qualification and NDE for cross-country pipelines.
Key requirements (7)
  • Welding of pipelines and related facilities.
  • Procedure qualification: PQR with mechanical tests (tensile, bend, nick break, Charpy where required).
  • Welder qualification by destructive bend tests OR radiography.
  • Acceptance criteria (Section 9): undercut, porosity, slag, lack of fusion limits.
  • RT acceptance: porosity max 1/8" diameter, max 4 in 6" weld length (general).
  • Automated UT (AUT) increasingly used for offshore / large diameter.
  • Annex A: alternative acceptance via fitness-for-purpose (ECA).
API 5L / ISO 3183 Specification for Line Pipe
Edition: 46th Edition (2018, A2 2024) Region: USA, Global
Line pipe for pipeline transportation systems (PSL 1 / PSL 2).
Key requirements (7)
  • Line pipe specification — PSL 1 (basic) and PSL 2 (more stringent toughness, hydrostatic, NDT).
  • Grades: B, X42, X46, X52, X56, X60, X65, X70, X80 (number = SMYS in ksi).
  • Manufacturing: SMLS (seamless), HFW (high-frequency welded ERW), SAWL (submerged arc longitudinal), SAWH (helical SAW).
  • Hydrotest: per pipe (each length) — 75% SMYS minimum × 10 sec.
  • Charpy impact for PSL 2: minimum average 27 J at 0 °C body, 40 J for ≥ X65.
  • DWTT (drop weight tear test) for ≥ X70 in PSL 2.
  • Heat treatment: hot rolled, normalized, Q&T per grade.

ASME — 1 standard

ASME B31.4 Pipeline Transportation Systems for Liquids and Slurries
Edition: 2022 Edition Region: USA, Global
Cross-country and in-plant pipelines for liquid hydrocarbons, LPG, ammonia, alcohols, slurries, etc.
Key requirements (7)
  • Liquid hydrocarbon pipelines (crude, product, NGL, anhydrous ammonia, alcohols).
  • Design factor F: 0.72 (general) — yield stress × 0.72 = max design hoop stress.
  • Hydrotest: 1.25 × design pressure for 4 hours minimum.
  • Pipe per API 5L; welding per API 1104.
  • External coating per NACE SP0169; cathodic protection per NACE SP0169.
  • MAOP = 2·S·t·F·E·T / D where E = joint factor, T = temp factor.
  • Class location concept used in B31.8 not B31.4 (B31.4 single design factor).

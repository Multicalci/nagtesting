API 609

Butterfly Valves: Double-Flanged, Lug- and Wafer-Type

Issuing body
API
Latest edition
8th Edition (2021)
Region
USA, Global
Equipment items
1

Scope

Butterfly valves Class 150 through 600, lug, wafer and double-flanged designs.

Key requirements (6)

  • Butterfly valves — Category A (concentric soft-seated, ANSI Class 125/150) and Category B (high-performance double or triple offset).
  • Cat A: utility / general service; rubber-lined or PTFE-lined.
  • Cat B (double-offset): metal- or soft-seated; bidirectional zero-leakage.
  • Cat B triple-offset: torque-seated, fire-safe, high-temp.
  • Hydrotest per API 598.
  • Wafer or lugged body styles.

API 617

Axial and Centrifugal Compressors and Expander-Compressors

Issuing body
API
Latest edition
9th Edition (2022)
Region
USA, Global
Equipment items
2

Scope

Centrifugal and axial compressors plus expander-compressors for petroleum, petrochemical and gas industries.

Key requirements (10)

  • Centrifugal and axial compressors — 8th edition restructured into 5 parts.
  • Part 1 General; Part 2 Centrifugal/axial; Part 3 Integrally geared; Part 4 Expander-compressors; Part 5 Packagers.
  • Design pressure: max suction + 110% × (max disch − min suct) — covers settle-out.
  • Hydrotest: 1.5 × max allowable working pressure of casing.
  • Mechanical run test: minimum 4 hours at MCS (max continuous speed) on test stand.
  • Performance test per ASME PTC 10 — Class 1 (full performance) or Class 2 (verification).
  • Vibration: API 670 — bearing housing < 7.6 mm/s; shaft displacement < 25 µm + 12,000/N (rpm) µm.
  • Aerodynamic stall margin ≥ 10 % typical; surge protection mandatory (anti-surge controller).
  • Lube/seal oil systems per API 614.
  • Driver coupling per API 671.

Applicable Standards (4)

API — 3 standards

API 6D / ISO 14313 Pipeline and Piping Valves
Edition: 25th Edition (2021) Region: USA, Global
Ball, check, gate, and plug valves for pipeline service.
Key requirements (7)
  • Pipeline valves — ball, gate, check, plug for liquid and gas pipelines.
  • Pressure classes: ASME B16.34 (150–2500) plus API-specific (10000 / 15000 psi).
  • Full-bore (full opening) — passes pipeline pig with min restriction.
  • Fire-safe per API 6FA / ISO 10497.
  • Anti-static device required (ball/stem ground continuity).
  • Double Block & Bleed (DBB) and Double Isolation & Bleed (DIB) configurations.
  • Cavity over-pressure relief required (self-relieving seat or cavity vent).
API 6FA Fire Test for Valves
Edition: 4th Edition (2020) Region: USA, Global
Fire-type testing of valves to demonstrate fire-safe performance.
API 622 Type Testing of Process Valve Packing for Fugitive Emissions
Edition: 3rd Edition (2018) Region: USA, Global
Type testing of valve stem packing material for fugitive emission performance (low-E packing).
Key requirements (7)
  • Type testing of process valve packing for fugitive emissions.
  • Test temperature: 260 °C (500 °F) typical.
  • Test cycles: 1510 mechanical cycles + 5 thermal cycles.
  • Leakage acceptance: ≤ 100 ppmv methane (or per spec).
  • Materials qualified: graphite, PTFE, flexible-graphite die-formed.
  • Mating shaft surface finish: typically Ra ≤ 0.4 µm (16 µin).
  • Drives selection of low-emission packing for VOC service.

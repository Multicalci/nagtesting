Bolted Tanks for Storage of Production Liquids

API
Latest edition

Scope

Bolted upstream production storage tanks.

Key requirements (7)

  • Bolted production storage tanks for upstream oil & gas operations.
  • Capacity range: typically up to 10,000 bbl.
  • Bolted assembly — easier to relocate than welded.
  • Material: carbon steel; coating/lining per service.
  • Design pressure: atmospheric (gauge oz/in² range).
  • Roof: deck or cone; floor flat-bottomed.
  • Used for crude, produced water, brine in field operations.

API 672

Packaged, Integrally Geared Centrifugal Air Compressors for Petroleum, Chemical, and Gas Industry Services

Issuing body
API
Latest edition
5th Edition (2019)
Region
USA, Global
Equipment items
0

Scope

Packaged plant air compressors (multistage, integrally geared, motor driven).

Key requirements (8)

  • Packaged integrally geared centrifugal air compressors.
  • Pressure range: typically up to 50 bar discharge.
  • Hydrotest: 1.5 × MAWP.
  • Mechanical run test: 4 hours at rated speed.
  • Performance test per ASME PTC 10.
  • Vibration per API 670.
  • Pinion vibration limits stricter than driver shaft.
  • Intercooler approach typically ≤ 8 °C.

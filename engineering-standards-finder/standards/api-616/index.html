API 616

Gas Turbines for the Petroleum, Chemical, and Gas Industry Services

Issuing body
API
Latest edition
5th Edition (2011, R2020)
Region
USA, Global
Equipment items
1

Scope

Open, simple- and regenerative-cycle combustion gas turbines for industrial drive service.

Key requirements (9)

  • Gas turbines for petroleum, chemical, gas industry service.
  • Power range: typical 1–250+ MW.
  • Performance test per ASME PTC 22.
  • Mechanical run test: 4 hours at MCS.
  • Allowable inlet ΔP: typically ≤ 250 mmH2O total.
  • Allowable exhaust backpressure: typically ≤ 350 mmH2O.
  • NOx limit: dictated by site emissions permit; DLE/DLN combustors typical.
  • Vibration per API 670; instrumentation mandatory.
  • Gas fuel filtration: ≤ 5 µm absolute typical.

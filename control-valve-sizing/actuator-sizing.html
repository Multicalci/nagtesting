Method & traceability

  • Required seating thrust F = F_unbalance + F_packing + F_seat, the standard sliding-stem actuator-sizing decomposition (Fisher Control Valve Handbook; ISA/ANSI practice).
  • Static unbalance F_unbalance = A_unbal · ΔP. Unbalanced trim: A_unbal = A_port − A_stem; balanced trim: ≈ 0.10 · A_port residual.
  • Packing friction scales with stem diameter, packing type and pressure class — a typical engineering estimate, not a vendor-guaranteed value. Confirm against the packing manufacturer's data.
  • Seat load F_seat = seat-load · π · d_port, seat-load per lineal inch of port circumference set by the FCI 70-2 leakage class (soft < metal IV < metal V).
  • Available thrust: double-acting P·A; air-to-close P·A − F_spring; air-to-open uses the spring bench-set to seat. Sign convention is worst-case for closure.
  • Scope: sliding-stem (globe/angle) valves. Rotary valves (ball/butterfly) require a torque check — a separate calculation not covered here.

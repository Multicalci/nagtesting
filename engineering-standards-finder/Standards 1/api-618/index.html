Reciprocating Compressors for Petroleum, Chemical, and Gas Industry Services

Scope

Reciprocating compressors for severe process service.

Key requirements (9)

  • Reciprocating compressors for petroleum, chemical, gas service — 5th edition.
  • Service classification by speed: low-speed (<330 rpm), moderate (330–800), high-speed (>800).
  • Pulsation control: API 618 'design approach' M2 (analog), M3 (acoustical), M4 (mechanical+acoustical).
  • Pulsation limits: ≤ 1 % peak-to-peak of average line pressure, OR ≤ 7 kPa (whichever less).
  • Crank shaft web deflection: < 0.0005 × stroke (TIR).
  • Hydrotest: 1.5 × MAWP for cylinders & all pressure-containing parts.
  • Mechanical run test: 4-hr continuous at rated speed and load.
  • Valves typically replaced at 8,000–25,000 hrs; valve lift ≤ 3 mm typical.
  • Lube oil consumption rate: 0.06–0.25 mL/kWh typical for distance-piece-lubricated.

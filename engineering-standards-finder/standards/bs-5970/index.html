BS 5970

Code of practice for thermal insulation of pipework, ductwork, vessels and equipment in the temperature range −100 °C to +870 °C

Issuing body
BSI
Latest edition
2012
Region
UK, Global
Equipment items
7

Scope

Application code for hot and cold thermal insulation in industrial service.

Key requirements (6)

  • Code of practice for thermal insulation of pipework, ducts, vessels.
  • Covers hot insulation, cold insulation, personnel protection, frost protection.
  • Thickness selection charts based on heat loss / surface temperature targets.
  • Vapor barrier requirements for cold service detailed.
  • Cladding / metal jacketing specifications.
  • Personnel protection target: surface temperature ≤ 60 °C.

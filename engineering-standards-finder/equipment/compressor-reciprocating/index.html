API — 2 standards

API 618 Reciprocating Compressors for Petroleum, Chemical, and Gas Industry Services
Edition: 5th Edition (2007, R2018) Region: USA, Global
Reciprocating compressors for severe process service.
Key requirements (9)
  • Reciprocating compressors for petroleum, chemical, gas service — 5th edition.
  • Service classification by speed: low-speed (<330 rpm), moderate (330–800), high-speed (>800).
  • Pulsation control: API 618 'design approach' M2 (analog), M3 (acoustical), M4 (mechanical+acoustical).
  • Pulsation limits: ≤ 1 % peak-to-peak of average line pressure, OR ≤ 7 kPa (whichever less).
  • Crank shaft web deflection: < 0.0005 × stroke (TIR).
  • Hydrotest: 1.5 × MAWP for cylinders & all pressure-containing parts.
  • Mechanical run test: 4-hr continuous at rated speed and load.
  • Valves typically replaced at 8,000–25,000 hrs; valve lift ≤ 3 mm typical.
  • Lube oil consumption rate: 0.06–0.25 mL/kWh typical for distance-piece-lubricated.
API 670 Machinery Protection Systems
Edition: 5th Edition (2014, R2021) Region: USA, Global
Vibration, axial-position, bearing temperature monitoring for rotating machinery.
Key requirements (7)
  • Machinery protection systems for rotating equipment.
  • Standard sensors: 8 mm proximity probes (eddy current), velocity sensors, accelerometers, KPH probes for speed.
  • Signal: 200 mV/mil sensitivity for proximity probes (5 V across 25 mil gap).
  • Standard alert/shutdown: 25 µm Alert / 50 µm Trip on shaft displacement (typical, tunable).
  • Bearing temperature RTD: 100 Ω Pt; 110 °C Alert / 120 °C Trip typical.
  • Casing vibration (accelerometer / velocity): 7.6 mm/s RMS typical alert for centrifugal machines.
  • API 670 monitoring system has 2-out-of-3 voting for trips, with bypass keys for maintenance.

ASME — 1 standard

ASME PTC 10 Performance Test Code on Compressors and Exhausters
Edition: 1997 (R2014) Region: USA, Global
Performance test procedures for centrifugal and axial compressors.
Key requirements (7)
  • Performance test code on compressors and exhausters.
  • Two test classes: Class 1 (full load, design fluid), Class 2 (model/equivalent fluid).
  • Test parameters: inlet pressure/temperature, discharge pressure/temperature, flow, power.
  • Polytropic efficiency reported (not isentropic) — preferred for multistage.
  • Test points: minimum 4 spanning surge to choke.
  • Uncertainty: typically ±2–4% on guaranteed performance.
  • Used to verify API 617/618/619 compressor compliance.

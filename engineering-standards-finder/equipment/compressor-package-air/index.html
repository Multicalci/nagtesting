Applicable Standards (3)

API — 1 standard

API 619 Rotary-Type Positive-Displacement Compressors for Petroleum, Petrochemical and Natural Gas Industries
Edition: 5th Edition (2010, R2017) Region: USA, Global
Dry and oil-flooded rotary-type positive-displacement compressors (mainly screw type).
Key requirements (6)
  • Rotary positive-displacement compressors: dry-screw, oil-flooded screw, lobe.
  • Hydrotest: 1.5 × MAWP for casing and pressure parts.
  • Mechanical run test: 4-hr continuous at rated speed.
  • Performance test per ISO 1217 or ASME PTC 9.
  • Vibration per API 670.
  • Inlet pulsation analysis required when in pulsing service.

Edition: 1970 (R1997) Region: USA, Global
Performance testing of reciprocating and rotary positive-displacement compressors.

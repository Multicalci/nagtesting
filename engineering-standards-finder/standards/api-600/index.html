API 600

Steel Gate Valves — Flanged and Butt-Welding Ends, Bolted Bonnets

Issuing body
API
Latest edition
14th Edition (2021)
Region
USA, Global
Equipment items
1

Scope

Bolted-bonnet steel gate valves with flanged or butt-weld ends, NPS 2 and larger. Identical to ISO 10434.

Key requirements (6)

  • Bolted-bonnet steel gate valves for petroleum and natural gas.
  • Designs: outside-screw-and-yoke (OS&Y) rising stem, wedge or parallel-slide gate.
  • Wall thickness ≥ ASME B16.34 plus API 600 Table 1 minimums (more conservative).
  • Trim numbers: API Trim 5 (13Cr stem/disc), Trim 8 (Stellite-faced), etc. per Table H.
  • Hydrotest per API 598 (shell, backseat, low-pressure air seat, high-pressure water seat).
  • Body-bonnet joint: ring joint (RTJ) or pressure-sealed bonnet for higher classes.

Applicable Standards (10)

API — 3 standards

API 610 / ISO 13709 Centrifugal Pumps for Petroleum, Petrochemical and Natural Gas Industries
Edition: 12th Edition (2021) Region: USA, Global
Workhorse standard for heavy-duty process centrifugal pumps. Pump types OH1–OH6, BB1–BB5, VS1–VS7.
Key requirements (13)
  • Pump types: OH (overhung), BB (between bearings), VS (vertically suspended) — 13 configurations.
  • OH1 = foot-mounted, OH2 = centerline-mounted, OH3 = vertical inline rigid coupling.
  • BB1 = axially-split single-stage, BB2 = radially-split, BB3 = axially-split multistage, BB5 = barrel (double-casing) multistage.
  • VS = vertically suspended (sump, can-type, etc.) — VS1 to VS7.
  • Minimum design pressure: max suction + 1.5 × differential, or maximum discharge — whichever is greater.
  • Hydrostatic test: 1.5 × maximum allowable working pressure, ≥ 30 minutes.
  • L10 bearing life: ≥ 25,000 hrs continuous operation; ≥ 16,000 hrs at maximum loads.
  • Shaft deflection: ≤ 50 µm (0.002 in) at primary seal faces under maximum hydraulic loads.
  • Allowable nozzle loads per Table 5 (forces and moments on suction & discharge).
  • Performance test per ISO 9906 Grade 2B (typical).
  • Vibration limits per API 670 — overall < 3 mm/s RMS for new pumps typically.
  • Materials: Annex H — material classes A (CI), B (CS), C (12Cr), D (austenitic SS), S (duplex), etc.
  • Sealing: per API 682 (mechanical seal piping plans 11, 13, 21, 23, 32, 52, 53A/B/C, 54).
Note: Default for refinery, petrochemical and oil & gas pumps.
API 682 / ISO 21049 Pumps — Shaft Sealing Systems for Centrifugal and Rotary Pumps
Edition: 4th Edition (2014, R2020) Region: USA, Global
Mechanical seal designs and seal piping/flushing plans (Plans 11, 13, 21, 23, 32, 52, 53A/B/C, 54, 62, 75, 76).
Key requirements (8)
  • Mechanical seal categories: Cat 1 (general), Cat 2 (refinery medium), Cat 3 (refinery severe).
  • Seal types: Type A (pusher), Type B (metal bellows), Type C (metal bellows hi-temp).
  • Arrangements: 1 = single seal, 2 = dual unpressurized (tandem), 3 = dual pressurized (double).
  • Standard piping plans: Plan 11 (flush from discharge), 13 (return to suction), 21 (cooler in flush), 23 (cooler in seal chamber pumping ring), 32 (external clean flush), 52 (unpressurized buffer), 53A/B/C (pressurized barrier), 54 (external pressurized barrier), 62 (quench).
  • Plan 53B: pressurized bladder accumulator instead of nitrogen-pressurized pot.
  • Plan 53C: piston accumulator referenced to discharge pressure.
  • Seal chamber dimensions standardized for 25/40/50/80/100 mm shafts.
  • Default qualification testing: 100-hour test on each seal series.
API 685 Sealless Centrifugal Pumps for Petroleum, Petrochemical, and Gas Industry Service
Edition: 3rd Edition (2021) Region: USA, Global
Magnetic-drive and canned-motor sealless pumps for heavy-duty service.
Key requirements (8)
  • Sealless centrifugal pumps (mag-drive and canned-motor) for petroleum, petrochemical, gas.
  • Companion to API 610 — same hydraulic and mechanical principles.
  • Pump types: OH and BB configurations only (no VS).
  • Hydrotest: 1.5 × MAWP for ≥ 30 minutes.
  • L10 bearing life: ≥ 25,000 hrs at design conditions.
  • Containment shell: full pressure rating required (canister leakage = catastrophic).
  • Mandatory monitoring: power monitor, bearing temp, leakage detection.
  • Used for hazardous, toxic, or leak-sensitive services where seal failure unacceptable.

Edition: 1977 (R2018) Region: India
Indian special-purpose pump requirements.
Key requirements (6)
  • Technical requirements for rotodynamic special purpose pumps.
  • Indian counterpart of API 610 (less stringent).
  • Hydrotest: 1.5 × maximum allowable working pressure.
  • Performance test: per IS 5120 / IS 9137 / ISO 9906.
  • Materials per IS 5120 Table — class I (CI), II (CS), III (12Cr), IV (SS), V (duplex).
  • Used for refinery, petrochemical pumps when API 610 not specified.

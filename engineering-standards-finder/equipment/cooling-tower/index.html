Edition: 2019 Region: USA, Global
Cooling tower thermal performance acceptance testing.
Key requirements (6)
  • Thermal performance acceptance test code for cooling towers.
  • Test parameters: water flow, hot/cold water temperatures, wet-bulb temperature, fan power.
  • Test conditions must be within design tolerance: typically ±10% of design flow, WB within ±15 °F (8 °C) of design.
  • Capability = (Q_test/Q_design) × 100 % at corrected conditions; ≥ 95 % typical guarantee.
  • Tower characteristic curve (KaV/L) developed from test data (Merkel/Poppe method).
  • Drift loss: ≤ 0.005 % of circulating water typical for modern drift eliminators.

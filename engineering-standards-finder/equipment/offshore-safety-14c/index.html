API — 2 standards

API RP 14C Analysis, Design, Installation, and Testing of Safety Systems for Offshore Production Facilities
Edition: 8th Edition (2017) Region: USA, Global
SAFE chart methodology for offshore process safety system design.
Key requirements (7)
  • Analysis, design, installation and testing of safety systems for offshore production platforms.
  • Safety Analysis Function Evaluation (SAFE) chart approach.
  • Two levels of protection: primary (process control) + secondary (shutdown).
  • ESD (Emergency Shutdown) levels: Tier 1 (process), Tier 2 (platform).
  • Mandatory safety devices: PSH, PSL, LSH, LSL, TSH, USV, BSDV.
  • Required testing intervals: monthly to quarterly depending on device.
  • Companion to API 754 for performance metrics.
API RP 754 Process Safety Performance Indicators for the Refining and Petrochemical Industries
Edition: 2nd Edition (2021) Region: USA, Global
Tier 1–4 leading and lagging process safety indicators (PSE, LOPC, etc.).

IEC — 1 standard

IEC 61511 Functional safety — Safety instrumented systems for the process industry sector
Edition: 2016 + A1 2017 Region: Global
SIS lifecycle, SIL, application of IEC 61508 in process industries.
Key requirements (9)
  • Safety Instrumented Systems for the process industry.
  • Safety lifecycle: H&RA → SIL allocation → design → operate → modify → decommission.
  • SIL levels (by PFDavg): SIL 1 (10⁻²–10⁻¹), SIL 2 (10⁻³–10⁻²), SIL 3 (10⁻⁴–10⁻³), SIL 4 (10⁻⁵–10⁻⁴).
  • Risk reduction factor RRF = 1/PFDavg.
  • SIL determination: LOPA, risk graph, risk matrix.
  • Architectures: 1oo1, 1oo2, 2oo3, 2oo4 — voting + redundancy for SIL.
  • Proof test interval (PTI) drives PFDavg: PFDavg ≈ λDU·PTI/2 for 1oo1.
  • MooN voting requires hardware fault tolerance (HFT) per Tables 3/4.
  • Cybersecurity: refer to IEC 62443 for SIS.

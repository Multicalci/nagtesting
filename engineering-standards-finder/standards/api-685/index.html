API 685

Sealless Centrifugal Pumps for Petroleum, Petrochemical, and Gas Industry Service

Issuing body
API
Latest edition
3rd Edition (2021)
Region
USA, Global
Equipment items
2

Scope

Magnetic-drive and canned-motor sealless pumps for heavy-duty service.

Key requirements (8)

  • Sealless centrifugal pumps (mag-drive and canned-motor) for petroleum, petrochemical, gas.
  • Companion to API 610 — same hydraulic and mechanical principles.
  • Pump types: OH and BB configurations only (no VS).
  • Hydrotest: 1.5 × MAWP for ≥ 30 minutes.
  • L10 bearing life: ≥ 25,000 hrs at design conditions.
  • Containment shell: full pressure rating required (canister leakage = catastrophic).
  • Mandatory monitoring: power monitor, bearing temp, leakage detection.
  • Used for hazardous, toxic, or leak-sensitive services where seal failure unacceptable.

API 613 Special-Purpose Gear Units for Petroleum, Chemical and Gas Industry Services
Edition: 5th Edition (2003, R2018) Region: USA, Global
Single- and double-helical, parallel-shaft, one- or two-stage speed-increasing or -reducing gear units.
Key requirements (8)
  • Special-purpose gear units for petroleum, chemical, gas service.
  • Service factor: minimum 1.4 (API minimum) — typical 1.5–1.8.
  • Single-helical or double-helical (herringbone) tooth form.
  • AGMA quality grade Q12 minimum (high precision).
  • Mechanical run test: 4 hours at maximum continuous speed.
  • Lube oil: API 614 system; oil flow ≥ 4× heat removal need at design.
  • Vibration limits per API 670.
  • Backlash: minimum specified to prevent excessive thermal seizure.
API 671 / ISO 10441 Special-purpose Couplings for Petroleum, Chemical and Gas Industry Services
Edition: 5th Edition (2020) Region: USA, Global
Disc-pack couplings for high-speed special-purpose machinery.
API 670 Machinery Protection Systems
Edition: 5th Edition (2014, R2021) Region: USA, Global
Vibration, axial-position, bearing temperature monitoring for rotating machinery.
Key requirements (7)
  • Machinery protection systems for rotating equipment.
  • Standard sensors: 8 mm proximity probes (eddy current), velocity sensors, accelerometers, KPH probes for speed.
  • Signal: 200 mV/mil sensitivity for proximity probes (5 V across 25 mil gap).
  • Standard alert/shutdown: 25 µm Alert / 50 µm Trip on shaft displacement (typical, tunable).
  • Bearing temperature RTD: 100 Ω Pt; 110 °C Alert / 120 °C Trip typical.
  • Casing vibration (accelerometer / velocity): 7.6 mm/s RMS typical alert for centrifugal machines.
  • API 670 monitoring system has 2-out-of-3 voting for trips, with bypass keys for maintenance.

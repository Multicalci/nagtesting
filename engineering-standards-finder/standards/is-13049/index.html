Diaphragm-actuated, Single Seated Top Guided Globe Type Pneumatic Control Valves

Scope

Indian control valve standard.

Key requirements (6)

  • Specification for diaphragm-operated control valves.
  • Indian counterpart of ISA 75 / IEC 60534 family.
  • Cv calculation, leakage classes (I–VI), inherent characteristics.
  • Hydrotest per IS 4733 / API 598.
  • Pressure classes per IS 4853 (PN 16 to PN 250 typical).
  • Used for compliance in Indian refineries and process plants.

API 624

Type Testing of Rising Stem Valves Equipped with Graphite Packing for Fugitive Emissions

Issuing body
API
Latest edition
2nd Edition (2018)
Region
USA, Global
Equipment items
1

Scope

Fugitive emission type testing for rising-stem valves with graphite packing.

Key requirements (7)

  • Type testing of rising-stem valves equipped with graphite packing for fugitive emissions.
  • Test temperature: 260 °C; pressure: 600 psi typical.
  • Cycles: 310 mechanical + 3 thermal.
  • Leakage acceptance: ≤ 100 ppmv methane.
  • Applies to gate, globe, parallel-slide and similar valves.
  • References API 622 packing requirements.
  • Used to qualify entire valve assembly, not just packing.

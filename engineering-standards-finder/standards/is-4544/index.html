Scope

Indian code for safe handling and storage of ammonia (anhydrous and aqueous).

Key requirements (5)

  • Indian Code of Safety for Ammonia.
  • Storage classifications matching CGA G-2.
  • Mandatory PWHT for carbon steel ammonia vessels.
  • Distance / separation requirements per Indian Petroleum & Explosives Safety Org. (PESO) approval.
  • Loading/unloading procedures, PPE, emergency response covered.

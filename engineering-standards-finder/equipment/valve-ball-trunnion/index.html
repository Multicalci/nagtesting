Applicable Standards (6)

API — 4 standards

API 6D / ISO 14313 Pipeline and Piping Valves
Edition: 25th Edition (2021) Region: USA, Global
Ball, check, gate, and plug valves for pipeline service.
Key requirements (7)
  • Pipeline valves — ball, gate, check, plug for liquid and gas pipelines.
  • Pressure classes: ASME B16.34 (150–2500) plus API-specific (10000 / 15000 psi).
  • Full-bore (full opening) — passes pipeline pig with min restriction.
  • Fire-safe per API 6FA / ISO 10497.
  • Anti-static device required (ball/stem ground continuity).
  • Double Block & Bleed (DBB) and Double Isolation & Bleed (DIB) configurations.
  • Cavity over-pressure relief required (self-relieving seat or cavity vent).
API 6FA Fire Test for Valves
Edition: 4th Edition (2020) Region: USA, Global
Fire-type testing of valves to demonstrate fire-safe performance.
API 608 Metal Ball Valves — Flanged, Threaded and Welding Ends
Edition: 6th Edition (2021) Region: USA, Global
Floating and trunnion-mounted metal ball valves, Class 150–600.
Key requirements (5)
  • Metal ball valves — flanged, threaded, welded ends.
  • Soft-seated (PTFE / RPTFE / PEEK) or metal-seated.
  • Pressure rating per ASME B16.34.
  • Hydrotest per API 598.
  • Floating ball (smaller sizes) or trunnion-mounted (larger sizes).
API 598 Valve Inspection and Testing
Edition: 10th Edition (2016) Region: USA, Global
Inspection, examination, supplementary examinations and pressure-test requirements for valves.
Key requirements (5)
  • Valve inspection and testing — covers the test requirements referenced by other valve standards.
  • Tests: shell (high-pressure water), backseat (where applicable), low-pressure pneumatic seat, high-pressure hydrostatic seat, optional high-pressure pneumatic seat.
  • Allowable leakage: zero for resilient-seated; small allowable rate for metal-seated (Table 6).
  • Test duration: minimum hold times by valve size.
  • Visual examination during pressurization.
